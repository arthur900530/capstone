# Skill Content

This skill details how to perform a Legal Entity Identifier (LEI) search using the Global Legal Entity Identifier Foundation (GLEIF) website. This process helps in Know Your Customer (KYC) analysis by providing detailed information about a company's legal status and structure.

**Step-by-step Guide:**

1.  **Open browser and navigate to GLEIF website**: Open a web browser and go to `https://search.gleif.org`.
2.  **Type company name into search bar**: In the "Search LEI Records" section, enter the name of the company you are interested in (e.g., "nvidia") into the search bar.
3.  **Select relevant company from options**: From the list of search results, select the entry that corresponds to the company you are researching (e.g., "NVIDIA Corporation").
4.  **Locate LEI code**: Once the company's detailed page loads, find the LEI code. This code is typically displayed at the very top of the company's information.
