# Excel Worksheet Management

This skill covers the fundamental actions required to manage the structure of an Excel workbook through its worksheets.

## Key Actions

### Navigation
- **Direct Selection:** Click on the worksheet tabs at the bottom of the Excel window to switch between them.
- **Scrolling Tabs:** Use the left and right arrow icons located to the left of the worksheet tabs to scroll through the list if there are more tabs than can be displayed.
- **Quick Jump (List View):** Right-click the arrow icons (navigation buttons) at the bottom left to open an 'Activate' dialog box. This provides a vertical list of all worksheets for faster navigation in large workbooks.

### Modification
- **Adding Worksheets:** Click the **+** (New sheet) icon next to the existing worksheet tabs to insert a blank sheet.
- **Deleting Worksheets:** Right-click on a specific worksheet tab and select **Delete** from the context menu. Caution: Deleting a sheet with data may prompt a permanent deletion warning.
- **Renaming (Context):** While not explicitly detailed, right-clicking a tab provides the "Rename" option to change the sheet's header.

## Best Practices
- Keep related data on separate sheets to maintain a clean workbook structure (e.g., "Summary Data", "Formulas", "Charts").
- Use the Right-click navigation menu when dealing with workbooks containing more than 10-15 sheets to avoid excessive scrolling.
